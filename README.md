## Setup 🛠️
### Frontend (Vue.js) 🖥️
- `yarn`
- `yarn dev`

### Backend (Python Flask) 🐍
- `cd server`
- `cp .env.sample .env` (to create server/.env)
- Fill in `.env`
- `python main.py`

### Prompt Testing (Promptfoo + Llama 3 + Groq) 🧪
- `cp .env.sample .env`
- Fill in `.env`
- `yarn`
- `yarn ptest`
- `yarn view`
